MYSHELL  Commands Manual                   mkdir

NAME
       mkdir - 创建一个或多个目录

SYNOPSIS
       mkdir [DIRECTORY]...

DESCRIPTION
       mkdir 命令用于创建一个或多个指定的目录,不可以建立多层目录.

OPTIONS
       无

EXAMPLES
       创建一个目录:
          mkdir my_directory
          如果目录存在
          mkdir my_directory/cat

       创建多个目录:
          mkdir dir1 dir2 dir3
